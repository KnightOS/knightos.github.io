Ready to start with the KnightOS SDK?

If you're on a Unix-like system (i.e. Linux, OS X, BSD, Cygwin), you can get the SDK simply like this:

wget http://www.knightos.org/install-sdk
chmod +x install-sdk
./install-sdk

See knightos --help after installation to see how it works.

Then you can head over to the docs to get started!

For more detailed setup instructions you can visit the wiki.
